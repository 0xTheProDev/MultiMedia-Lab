Introduction

 MIPS Simulator is a Free and Open Source Software that helps you execute your MIPS assembly instructions right from your terminal in Desktop PC. This is a cross-platform application, meaning runnable from all devices. The current version of the software supports instructions upto MIPS-IV standard enabling a virtual machine with Load-Store architecture.

Pre-requisites

  • A working PC with with at least Pentium Core Processor
  • Minimum of 256 Mb Physical Memory
  • Any Operating System with Command Line Interface
  • A Node Runtime Environment

How to Install

Installing Proper Node Environment
MIPS Simulator is supported on both 6 LTS version and current 8+ version.
  1. Visit nodejs.org and select Download option from the menu
  2. Choose your appropriate platform and download either of the above mentioned.
  3. Once downloaded, open the installer (on Windows) or execute (on MacOS or Linux) and follow.
  4. Once installation is complete, open command line and type 
    $ npm -v
    to ensure that installation is a success.
Note: We strongly encourage you to use powershell or install any bash alternatives on Windows.
Installing The Application

Open your terminal and execute the following: 

$ npm i -g mips-simulator

How to Use

The application was developed with simplicity in mind. So to run an assembly program, you just need to do the following: 

$ mipc <assembly_source_file>

Contributing for Developers

If you want to contribute to the project, you will be required to have a GitHub id and Git installed in your system. Then clone the source repository by: 

$ git clone https://github.com/BytesClub/MIPS-Simulator
Test the current state of the application by 
$ npm test
Create a new branch and start making changes. Create a pull request with your changes. Use continious testing and linting to make sure your code does not break anything.
